User Interface Prototype

This was a project I completed for my IST 110 class at Penn State. The project was to take an app that already existed and make a new feature for it. The app modified was Kakao Talk, which is a messenging app. The features added in this prototype is an "group food order" feature. The concept is that if youre in a group with your friends, someone can initiate a group order. Working similar to github there would be multiple restaurants to choose from and order from at once. Once the order was created everyone in the group would be able to edit the order at the same time and then once the order was submitted everyones order would go through together. The prototype is a mockup of what it would look like and offers very limited functionality. For this project I used the program proto.io.

The interactive parts of this prototype are the menu pullover in the top right corner from the chat page. From there the fork an knife icon will take you to the order page. You can view your order from the list icon in the top right corner and the list of places would be a variety of restaurants. The "Pizza Restaurant" icon is the one that is interactive. Then you would choose from their menu items and hit the plus icon to add it to your orders. From there you can view your order in the orders page and that is also where you would submit your order.
